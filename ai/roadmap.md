# Roadmap

## 1. Frontend & Agent Setup
Status: pending
Goal: Initialize the Vercel AI SDK project and basic chat interface with model.

## 2. MCP Server Foundation
Status: pending
Goal: Set up the MCP server and connect it to the agent.

## 3. Planning & Sequencing Logic
Status: pending
Goal: Implement the agent's ability to ingest requirements, and break them into specific tool actions.

## 4. Human-in-the-Loop Pipeline
Status: pending
Goal: Build the UI and state management for the approval, execution tracking, and verification pauses.

## 5. Salesforce Tool Implementation
Status: pending
Goal: Build and expose the actual Salesforce metadata creation and deployment tools via the MCP server.

## 6. Self-Correction Engine
Status: pending
Goal: Implement the agent loop to catch Salesforce API errors and autonomously attempt fixes before alerting the user.

## 7. Planning & Requirement Storage
Status: pending
Goal: Implement the agent's ability to analyze, structure, and sort requirements, map out the required tool actions, and store this execution plan in the database.

## 8. Human-in-the-Loop UI
Status: pending
Goal: Build the frontend interface to render the structured requirements and present the explicit execution approval controls to the user.

## 9. Execution & Self-Correction Loop
Status: pending
Goal: Implement the logic for the model to execute a requirement's actions, autonomously fix errors, and halt for user verification before progressing to the next requirement.
