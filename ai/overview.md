# Overview

## Product
An AI agent system that accelerates Salesforce configuration by translating natural language requirements directly into live metadata deployments.

## Target Users
Salesforce Developers, Consultants, Architects, and non-technical Stakeholders.

## Problem
Bridging the gap between business requirements and Salesforce implementation traditionally takes weeks of manual configuration; this system eliminates that bottleneck, speeding up delivery by 1000x.

## Core Features
- Automated requirement sequencing and dependency mapping.
- Step-by-step action plan generation mapping to discrete Salesforce operations.
- Human-in-the-loop approval and verification checkpoints.
- Autonomous error handling and self-correction during tool execution.

## Constraints
- Must enforce explicit user approval before executing any deployment action.
- Must halt and wait for manual user verification in the Salesforce org before proceeding to the next requirement.
