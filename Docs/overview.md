# Overview

## Product
Metaforce is an AI-powered platform that accelerates Salesforce development by enabling users to describe requirements in natural language and have them automatically planned, executed, and deployed to a Salesforce org.

## Target Users
- Salesforce developers
- Salesforce architects
- Anyone who wants to build and deploy Salesforce requirements without writing code manually

## Problem
Salesforce development is slow, repetitive, and requires deep platform knowledge. Developers and architects spend too much time on boilerplate metadata creation instead of solving business problems.

## Core Features
- Conversational requirement gathering via a shared AI chat interface
- Incremental requirement planning — add, remove, and refine before execution
- Step-by-step action execution with per-action approval and error recovery
- Salesforce org connection via a Chrome extension (no OAuth setup required)
- Team collaboration — multiple users can work within the same project and shared conversation
- Metadata reuse library — every created component is saved for future reuse

## Constraints
- POC scope is limited to custom objects and custom fields
- Users must have an existing Salesforce org; no platform-provisioned orgs
- One org per project
- One shared conversation per project
- AppExchange installation is out of scope
